# FreelanceFlow
A CLI-based management tool for freelancers to track tasks and finance.

## Features
* **Client Management:** Add, View, and Delete clients.
* **Workflow Tracking:** Update project status.
* **Financial Reports:** Real-time earnings calculation.
* **Data Persistence:** Auto-saves to JSON.

## How to Run
1.  Install Python.
2.  Run `python main.py`.
